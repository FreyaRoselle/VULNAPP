# Vulnerability Assessment Lab – Vulnerable Web Application

## Overview
An intentionally vulnerable web application created for academic vulnerability assessment practice and security testing in a controlled environment.

⚠️ Unsafe by design. For lab use only.

## Objectives
- Practice vulnerability identification  
- Perform security scanning using automated tools  
- Analyze common OWASP Top 10 weaknesses  

## Implemented Vulnerabilities
- SQL Injection  
- Cross-Site Scripting (XSS)  
- Remote Code Execution (RCE)  
- Broken Access Control  
- Command Injection  

## Environment
- OS: Kali Linux  
- Web Server: Apache  
- Backend: PHP  
- Database: MySQL  
- Port: 80  

## Tools Used
- Nmap  
- Nikto  
- Nessus  

## Assessment Scope
- Port and service discovery  
- Web server misconfiguration analysis  
- Application vulnerability detection  

## Disclaimer
This project is intended strictly for educational and authorized lab use. Do not deploy in production or public environments.
